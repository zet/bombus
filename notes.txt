внимание! это только пример движка, очень мало сделано
1. аккаунт прописываeтся в jad файле
2. несколько аккаунтов пока отключено. 
   до написания кода редактирования аккаунта
3. никаких зелёных и красных кнопок пока нет (почти)
4. при вставке смайлика нет отмены.
5. неполная обработка ошибок при подключении.
   при ошибке аутентификации просто виснет на Login

а теперь что есть:
в ростере - 
 up(2)/down(8)
 left(4)/right(6)=pgup/pgdn
 1/7=home/end
 5=fire - открыть chathistory. (+left soft key for m55)

 right soft key - меню.
 в меню работают только status и quit

в chat history:
 up(2)/down(8)
 left(4)/right(6)=pgup/pgdn
 1/7=home/end
 5=fire - открыть сообщение (+left soft key for m55)

 right soft key - меню.
 в меню работают все пункты

в message
 up(2)/down(8)
 left(4)/right(6)=pgup/pgdn
 1/7=home/end
 5=fire - выход (+left soft key for m55)

вставка смайлика:
 up(2)/down(8)
 left(4)/right(6)
 5=fire - выбор смайлика (+left soft key for m55)
