Требования

Для работы программы требуется телефон/смартфон с поддержкой Java MIDP-2.0, или цветной телефон Siemens с поддержкой MIDP-1.0. Для того чтобы узнать какой у вашего телефона Java MicroEdition Profile (MIDP) перейдите по этой ссылке: http://www.jbenchmark.com/result.jsp

Протестировано на следующих телефонах:
  • Siemens M55, C60, S65, M65, CX65
  • Alcatel OT756
  • Motorola E398
  • Samsung SGH-D500

Также необходимо настроить телефон на работу для работы с GPRS-Internet (GPRS-WAP не имеет никакого отношения к Bombus). Если вы можете сидеть в интернете с компьютера через телефон то скорее всего дополнительная настройка телефона не потребуется, кроме того что тайм-аут (время от начала простоя до завершения текущей сессии GPRS) должен быть не менее 200 секунд. Настройки GPRS вы можете узнать у своего оператора сотовой связи.

Добавление существующего/создание нового аккаунта на сервере jabber

Bombus является jabber-клиентом, общение с пользователями ICQ возможно только после подключения транспорта, поэтому не пытайтесь вводить в следующие поля свой UIN, пароль и в качестве сервера login.icq.com! Создать аккаунт ICQ из него невозможно по этой же причине, для общения через ICQ у вас уже должен быть зарегистрирован UIN.

Аккаунт - связка логин (имя пользователя) + пароль

В сети jabber каждый пользователь имеет свой идентификатор, JID (Jabber ID). Например: mozilla@jabber.ru, где mozilla это логин (имя пользователя), а jabber.ru - сервер, на котором существует данный аккаунт. Также в jabber возможно выходить в сеть с одного JIDа одновременно сразу с нескольких клиентов. Для того чтобы отличать их было введено такое понятие как ресурс. Это произвольная символьная метка. Например: mozilla@jabber.ru/Bombus. В данном JIDе "Bombus" это и есть ресурс.


После первого запуска программы вы увидите пустое окно Accounts. Вам нужно добавить/создать новый аккаунт для того чтобы начать общение. Команда меню New account открывает форму со следующими полями:

  • Username: имя пользователя jabber. При регистрации нового аккаунта на сервере (внизу формы есть галочка Register account) имя пользователя не должно быть занято.
  • Password: пароль
  • Server: сервер jabber. Наиболее известны следующие сервера:
    • jabber.ru (IP: 213.248.55.180, 213.248.55.181)
    • jabber.ttn.ru (IP: 213.24.85.3)
    • jabber.kiev.ua (IP: 193.125.78.117)
    • jabber.infos.ru (IP: 195.209.229.42)
    • jabber.snc.ru (IP: 212.6.0.100)
  • Server IP: IP-адрес сервера. Нужен только в том случае, если в настройках телефона не заполнены поля DNS. Если не заполнены поля DNS и IP, то процесс соединения с сервером может длиться достаточно длительное время.
  • Port: порт по которому идёт соединение. Изменяйте только тогда когда будете уверены что знаете что делаете (но лучше вообще не надо, станет только хуже)
  • Register account: если установить эту галочку то Bombus попытается создать новый аккаунт на вышеуказанном вами сервере. Если у вас есть существующий аккаунт на этом сервере то её ставить не нужно.
  • Resource: ресурс, по умолчанию Bombus
  • Account name: имя аккаунта, произвольное

После добавления/создания аккаунта Bombus пытается соединиться с сервером и войти в сеть. Если ему это не удалось и появилось сообщение Login failed, то это значит что неправильно введён логин или пароль. Если же появилось Failed, значит или неправильно указан сервер, или неправильно настроен GPRS, либо какие-то ещё проблемы со связью.
