SE k700i
создание подключения для жаба-приложений: 
меню -> 
  связь -> 
    передача данных -> 
      учетные записи -> 
        создать учетную запись согласно данных, 
        предоставляемых оператором связи. 
меню -> 
  связь -> 
    настройка java -> 
      профиль -> 
        создать профиль, 
        где указать подключение через созданную учетную запись, 
        а также ип адрес, логин и пароль, предоставленные оператором связи
